    题目2: 查询水果价格

    题目描述

    给定四种水果，分别是苹果（apple）、梨（pear）、桔子（orange）、葡萄（grape），单价分别对应为3.00元/公斤、2.50元/公斤、4.10元/公斤、10.20元/公斤。

    首先在屏幕上显示以下菜单：

    用户可以输入编号1~4查询对应水果的单价。当连续查询次数超过5次时，程序应自动退出查询；不到5次而用户输入0即退出；输入其他编号，显示价格为0。

    输入格式:

    输入在一行中给出用户连续输入的若干个编号。

    输出格式:

    首先在屏幕上显示菜单。然后对应用户的每个输入，在一行中按格式“price = 价格”输出查询结果，其中价格保留两位小数。当用户连续查询次数超过5次、或主动输入0时，程序结束。

    输出样例1:

    输入样例2:

    输出样例2:

    解题思路

    这个问题可以通过编写一个简单的循环程序来解决。

    1. 首先，定义水果名称和对应的单价。可以使用数组来存储这些信息。
    2. 显示菜单，列出水果选项（1到4）以及退出选项（0）。
    3. 使用循环来接收用户输入。在循环中，需要进行以下步骤： a. 提示用户输入水果的编号。 b. 根据用户输入的编号，检查是否在有效范围内（1到4）。 c. 如果编号有效，输出对应的水果价格，否则输出价格为0。 d. 记录用户查询次数，当连续查询次数达到5次或用户输入0时，退出循环。

    题目3: 求整数段和

    题目描述

    给定两个整数A和B，输出从A到B的所有整数以及这些数的和。

    输入格式：

    输入在一行中给出2个整数 $A$ 和 $B$ ，其中 $ -100 \leq A \leq B \leq 100 $，其间以空格分隔。

    输出格式：

    首先顺序输出从$A$到$B$的所有整数，每5个数字占一行，每个数字占5个字符宽度，向右对齐。最后在一行中按Sum = X的格式输出全部数字的和X。

    解题思路

    这道题目要求根据给定的整数范围A和B，输出从A到B的所有整数，并计算它们的和。

    题目4: 阶乘

    题目描述

    给你一个正整数，你能算出它的阶乘是多少吗？

    输入描述:

    若干行，每行一个整数 $ n $ 满足 $ 1 \leq n \leq 20 $。

    输出描述:

    按顺序输出每个整数的阶乘，一个结果占一行。

    输入样例1
    输出样例1
    输入样例2
    输出样例2

    解题思路

    阶乘是一个正整数的连乘，例如，5的阶乘表示为5!，等于5 × 4 × 3 × 2 × 1。

    1. 读取输入的整数n。
    2. 使用一个循环来计算n的阶乘。可以使用一个变量来迭代计算，初始值设置为1，然后从1乘到n，每次迭代将当前值乘以迭代变量。
    3. 输出计算得到的阶乘值。
    4. 重复上述步骤，直到读取所有的输入整数。

    这个程序循环读取输入的整数n，然后计算n的阶乘并输出结果。一旦输入结束，即读取到文件结束标志EOF，程序将退出。这样，它可以处理多行输入，每行一个整数，计算每个整数的阶乘。

    注：本题测试数据会超出int的上限，需使用long long

    1. INT_MAX = 2147483647
    2. LLONG_MAX = 9223372036854775807

    题目6: 动态二维数组的转置

    题目描述

    设计一个矩阵类Matrix（整型动态二维数组类）类中成员函数有：
    构造函数（创建m行n列的矩阵）；析构函数（释放动态申请数组空间）；

    读入函数ReadMatrix（创建m行n列的矩阵元素的值）；

    输出函数WriteMatrix（输出m行n列的矩阵元素的值，每个元素占5个字符宽度 ）；

    转置函数void Transpose(Matrix& B)（将当前对象矩阵转置为一个n行m列的B矩阵）。

    在main()函数中输入m,n的值创建m行n列的矩阵A，矩阵B是A的转置矩阵。

    输入格式:

    接下m行，每行读入n个整数（读入A矩阵）

    输出格式:

    先输出A矩阵
    后输出B矩阵（A的转置矩阵）

    解题思路

    这道题需要实现一个矩阵类 Matrix，其中包括构造函数、析构函数、读入函数 ReadMatrix、输出函数 WriteMatrix 以及转置函数 Transpose。

    思路如下：

    题目7: A-B

    题目描述

    本题要求你计算 $A−B$。不过麻烦的是，$A$ 和 $B$ 都是字符串 —— 即从字符串A中把字符串B所包含的字符全删掉，剩下的字符组成的就是字符串 $A−B$。

    输入格式：

    输入在2行中先后给出字符串 $A$ 和 $B$。两字符串的长度都不超过$10^4$，并且保证每个字符串都是由可见的ASCII码和空白字符组成，最后以换行符结束。

    输出格式：

    在一行中打印出 $A−B$的结果字符串。

    输入样例：

    输出样例：

    解题思路

    思路：

    1. 读取两行输入，分别为字符串A和字符串B。
    2. 创建一个空字符串，用于存储A−B的结果。
    3. 遍历字符串A的每个字符，对于每个字符，检查它是否在字符串B中出现。
    4. 如果字符不在字符串B中出现，将它添加到结果字符串中。
    5. 最终，结果字符串中将只包含字符串A中不在字符串B中出现的字符。
    6. 输出结果字符串。
