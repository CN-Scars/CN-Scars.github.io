输出格式:

如果该信号是安全的，输出一行Safe，否则输出Danger。

题目3: 小s喝可乐

题目描述

题目背景

小s很喜欢喝可乐，尤其是日本生可乐。小s的库存有$n$瓶可乐，他喜欢将可乐按照一定的顺序饮用，这样他可以获得更多的愉悦感。

小s在为可乐的饮用顺序排序时，会按顺序遵循如下的规则：

  1. 优先考虑保质期，如果某瓶可乐快要过期了，那么就优先饮用这瓶可乐

  2. 其次，如果这瓶可乐是日本生可乐，那么优先饮用这瓶可乐

  3. 最后，如果这瓶可乐是女朋友送的，那么小s会更加珍惜它，不会优先饮用

对于每瓶可乐，都有一个编号值$i (1, 2, \dots, n)$。如果有两瓶可乐距离过期的时间一样，都是（或都不是）生可乐，都是（或都不是）女朋友送的，则按照编号进行排序。

输入格式:

第一行，输入一个整数$n$ $(1 \leq n \leq 2 \times 10^{5})$，代表可乐的瓶数。

接下来$n$行，对于每一行输入，输入3个整数，为$d, c1, c2$。

$d$ $(1 \leq d \leq 100)$代表可乐距离过期剩下的天数

若$c1$为0，代表这瓶可乐是生可乐，反之亦然

若$c2$为1，代表这瓶可乐是小$s$的女朋友送的，反之亦然

每瓶可乐的编号值按照输入的顺序从$1$开始编号

输出格式:

输出共有一行，即按照排序规则排序后的可乐的编号值，优先饮用的先输出（编号值用空格隔开）

题目5: K个B

题目描述

给定一个长度为$n$仅含有字符$A$和字符$B$构成的字符串，再给定一个期望值$k$，你可以将字符串中包括第$i$个位置以及之前的所有字符都换成$A$或$B$，使得字符串中$B$的个数有$k$个。

你需要编写程序，求出可以达成期望的$i$，并给出用于替换的字符。

输入格式:

测试数据有$T$组$(1 \leq T \leq 100)$。

对于每组样例，第一行输入$n (1 \leq n \leq 100)$和$k (1 \leq k \leq n)$。

第二行，输入长度为$n$仅含有字符$A$和字符$B$构成的字符串$s$。

输出格式:

  • 若不需要进行替换，则输出一行$0$

  • 若需要进行替换，输出一行，先输出开始替换的位置$i$，然后再输出用于替换的字符，之间用空格隔开。

如果$i$有多种可能，输出最小的一种即可。

样例解释

  • 对于第一组样例，字符串$s$中已经有$2$个字符$B$了，故不需要进行替换操作，输出0。

  • 对于第二组样例，唯一的方法就是把第一个位置的字符替换为$B$，这样新的字符串中才能有$3$个$B$，即$\mathrm{AABAB} \rightarrow \underline{\mathrm{B}}\mathrm{ABAB}$，故输出1 B。

  • 对于第三组样例，替换方案中最小的$i$是$1$，即$\mathrm{BAA} \rightarrow \underline{\mathrm{A}}\mathrm{AA}$，故输出1 A。

解题思路

记录$A$和$B$在字符串$s$出现的位置，根据需要去覆盖A或者B。来增加或减少B的个数。
