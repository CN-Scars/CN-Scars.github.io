题目4: 异！物！ #2

题目描述

顶镇同学的房间里有个储物箱，他的储物箱有个特点，第一个物品只能放在箱子底部，往后的物品都只能堆在前一个物品的上方，取一次物品时只能从储物箱的最上方取。今天他想整理他的储物箱里面的东西（方便起见，用整数数字来替代这些物品）。他会把东西拿出来，也会把已经拿出来的东西放进去，也会拿他充满芝士的雪豹机器人检测是否有异物。检测到有异物的标准如下：

  • 若此时储物箱内的数字的最大值超过了限定的最大值，则表明有异物。

你需要编写程序来模拟他整理东西时的情景。

输入格式:

第一行，两个整数，$T (1 \leq T \leq 1 \times10^5), max (-2^{60} \leq max \leq 2^{60})$，代表顶镇操作的次数和限定的最大值。

  • Put X，表示把一个整数$X (-2^{60} \leq X \leq 2^{60})$放入储物箱中；

  • Check，表示检测储物箱中是否有异物。

题目5: 龙王

题目描述

Scars有一次获得了【龙王】标识，同时，他获得了【呼风唤雨】的技能。

在群聊中，一共有$n$个成员，编号为$1, 2, \dots, n$，，但Scars不知道的是这$n$个群员都是kiki使用影分身之术的分身。愤怒的Scars会使用$s$次技能，这个技能可以让编号为$k$的群员的头上降下$L$升的雨水。在施放完成$s$次技能后，他会对所有群员进行$q$次查询，对于第$i$次查询，他会查询编号在$[l_i, r_i]$内的群员头上的降水总量。

输入格式:

第一行，三个正整数，$n, s, q$，代表群员数量、使用技能的次数和查询的次数。
第二行，$n$个正整数，为$a_1, a_2, \dots, a_n$，依次表示$n$位群员头上的初始降水量。
接下来$s$行，每行两个正整数，$k, L$，代表每次施放技能时，让编号为$k$的群员的头上降下$L$升的雨水。
再接下来$q$行，每行两个正整数，$l, r$，代表每次查询的编号区间范围为$[l_i, r_i]$。

输出格式:

共$q$行，对应$q$次查询的结果，即第$i$行的结果是第$i$次查询的结果。

输出样例:

数据范围

题目6: 谁问你了？

题目描述

在视频网站上，有些视频的评论区会出现下图中的现象：

假设A用户问了B用户，而B用户一定会回复A用户：谁问你了？每位用户也可能会去问其他的用户，也可能不会去询问别的用户。只要用户A有向用户B提问，B进行了回答，那么用户A和用户B就认为是一个用户群体。当然，若有用户没有向其他的任何用户提问或被其他的任何用户提问，那么这个用户就自成一个用户群体。请你编写代码，找出一个视频的评论区到底有多少种用户群体，并分别求出它们的群体人数。

输入格式:

共$m + 1$行：

  • 第一行，两个整数$n, m$，$n$代表一个视频的评论区用户的总数，$m$代表一共有$m$个用户发起了提问，保证被提问的用户不会再向发起提问的用户提问，也不会向自己提问。
  • 接下来$m$行，每行两个正整数$A, B$，为了方便起见，$A$和$B$就用作用户编号，代表用户$A$向用户$B$发起了提问。

输出格式:

共两行：

  • 第一行，输出一个正整数，代表这个视频的的评论区有多少种用户群体。
  • 第二行，从小到大输出所有用户群体的用户数，中间用空格隔开，末尾不得有多余的空格。

输入样例:

输出样例:

说明

数据范围
$1 \leq A, B \leq n \leq 2 \times 10^5, 0 \leq m \leq \frac{n(n-1)}{2}$。

解题思路

一道求无向图（不存在重边和自环）联通块数量和大小的模板题，若A、B两个用户在一个群体内，可以看作在一个并查集中，对于每个点，把它们的根都求出来并放入set容器中去重即可知道联通块个数，并把并查集的根进行标记，把属于这个根的节点数量累加即可求得每个联通块的大小。
